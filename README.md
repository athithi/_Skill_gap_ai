# 🚀 PRH Skill Gap AI

An intelligent system that analyzes resumes and identifies missing skills based on job requirements — then bridges the gap by recommending personalized learning paths.

---

## 💡 Problem Statement

Many job seekers struggle to understand why they are not shortlisted.
This project helps by identifying **skill gaps** between a candidate’s resume and job descriptions, and provides **actionable recommendations** to improve.

---

## 🧠 Features

* 🔍 **Resume Analysis**
  Extracts and analyzes skills from resumes using NLP techniques.

* 📊 **Skill Gap Detection**
  Compares candidate skills with job requirements to find missing skills.

* 🎯 **Bridge the Gap (Learning Path)**
  Recommends courses, certifications, and resources for the top missing skills.

* 👥 **Dual User Mode (Planned/Optional)**

  * Job Seeker: Skill insights + learning roadmap
  * Recruiter: Candidate evaluation support

## 🛠️ How It Works

1. Upload or input a resume
2. Provide a job description
3. System extracts skills from both
4. Compares and identifies missing skills
5. Generates a **learning path** to bridge the gap
